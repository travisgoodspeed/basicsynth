Example Projects

Several example project and associated score files are located in the BasicSynth/Examples folder. These can be examined to get a general idea of how the synthesizer can be used.

Project Score(s) Output File Description
TestJig.bsprj jig.nl jig.wav A simple "jig" using an FM synthesis instrument to simulate fiddles and drums.
ARestingPlace.bsprj Bass.nl,Melody.nl,Punch.nl,Runner.nl ARestingplace.wav Based on the hymn "My Faith Has Found a Resting Place". FM synthesis using the MatSynth instrument. Demonstrates the use of the dynamic panning and effects properties of the MatSynth instrument.
SunflowerSonata.bsprj swflr.nl swflr.wav A small ensemble of synthetic bass, marimba, wind and brass instruments.
PrettyFlowers.bsprj prettyflowers.nl prettyflowers.wav A simple 12-bar blues piece using a combination of sampled drum sounds and synthetic bass, electric piano and guitar. The bass and guitar show examples of using pitch-bend effects. Drum samples are from FreeSound.org.
Whirledwind.bsprj WhirledWind.nl WhirledWind.wav An example of an "ambient" composition using numerous polyrhythmic loops. Instruments include a mixture of synthetic FM drone and percussion sounds combined with sampled drum sounds. Sampled drum sounds are from the Tam-tam drum set, part of the OLPC project.
Pulse.bsprj pulse*.nl pulse.wav Pulse incorporates a variety of FM synthesis instrument sounds with filtered noise sounds. A porition of the score is generated using a morse-code program and then included into the main score. Note how the score is divided into multiple files for convienience.
